# thesaurus

1. review_base.py - извлечение отзывов (количество = 7371) с Яндекс.Маркета при помощи сервиса market-scanner.ru/doc 
Отзывы хранятся в папке tools в .json и .xlsx форматах

2. clear_data.py - очистка текстов отзывов от посторонних символов: перевод в нижний регистр и к нормализованному виду (рекомендую -> рекомендовать)
Очищенные отзывы хранятся в папке data_preparation так же в двух форматах

3. word_extraction.py - извлечение частей речи (прилагательных, существительных, глаголов, наречий) и подсчет их частоты встречаемости во всех отзывах в целом
Неразмеченные слова хранятся в папке unallocated_words
